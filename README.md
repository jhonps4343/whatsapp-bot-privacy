# Políticas de Privacidad y Eliminación de Datos - X engine

## 1. Política de Privacidad
Este programa fue desarrollado con el único propósito de agilizar la respuesta a los compradores del negocio mediante la API de WhatsApp. No recopilamos, almacenamos ni compartimos ninguna información personal, números de teléfono ni contenidos de los mensajes con terceros. Todos los datos se procesan localmente en la aplicación del usuario.

## 2. Instrucciones para la Eliminación de Datos
Dado que esta aplicación no almacena datos de los usuarios ni de sus clientes en servidores externos, no existe ninguna base de datos de la cual borrar información. Si deseas dejar de utilizar el servicio, simplemente cierra la aplicación o desconecta el acceso de tu cuenta de Meta for Developers.
